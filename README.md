   → 공고 요건 대응표(Fit Matrix) + 7년 흐름 마스터 다이어그램 + CS 연구원 ↔ 본인 경력 매핑

4) 문서작성 역량 대표 샘플 (PDF, 8쪽)   ← 보조 자료
   → 학부 시절 HWP 정리자료의 일부 (문서화 습관 증거)

## 📌 핵심 요약

**메인 산출물 — 환자후기 AI 분석 보고서 12장**
가상 ABC 정형외과 환자 후기 100건을 7개 카테고리 × 3단계 감성으로 AI 분류하고,
무작위 20건 수동 검증으로 일치율 88% (카테고리 85% / 감성 90%)를 확인했습니다.
인사이트 3건과 CS 개선 우선순위 3가지 제안을 검증 지표(KPI) 형태로 정리했고,
한계 4가지(가상 데이터 / 표본 편향 / 단일 시점 / AI 분류 오차)를 별도 슬라이드로 명시했습니다.

**보조 자료 — 8쪽 문서작성 역량 대표 샘플**
한서대학교 항공운항학과 재학 시 항공기상·항공기 시스템·항행시설 등 복잡한 전공 내용을
HWP 정리자료로 구조화해 동기들과 공유했던 자료 중 문서화 방식이 잘 드러나는 일부를 선별·압축했습니다.
이 학습용 문서화 패턴이 한화오션 외부 기관 보고자료, 대치동 컨설팅 자료, 그리고 환자후기 AI 분석 보고서까지
같은 방식으로 이어졌습니다.
